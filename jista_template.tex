\section{Introduction}

In order to reach the highest level of quality, authors should comply
with the rules set out in this template.

The template will be returned to the author for reorganization of the
articles not prepared in accordance with the template. Returned
articles must be returned after they have been arranged in accordance
with the rules.

% ============================================================
\section{Preparation of the Manuscript}

Articles containing incorrect, inaccurate or incomprehensible
sentences shall be returned in writing to be arranged in the language.

Times New Roman font (10\,pt, single line spacing and two-sided)
should be used.

% ----
\subsection{Page Setup \normalfont\textit{(Sayfa Düzeni)}}

The page size should be set to A4 (210\,×\,297\,mm). Page margins
should be: Top 3\,cm, Bottom 2.5\,cm, Left 2\,cm, Right 2\,cm.

% ----
\subsection{Paper Title and Abstract \normalfont\textit{(Başlık ve Özet)}}

The title of the article should be 18\,pt in Times New Roman font,
bold and the first letters shall be large, with a space of 1.5 lines
left, 18\,pt before and 6\,pt after.

% ----
\subsection{Section Titles \normalfont\textit{(Bölüm Başlıkları)}}

Section headings should be typed in Times New Roman 12\,pt, bold,
first letters large, single line spacing, 12\,pt before and 6\,pt
after.

% ----
\subsection{Subsection Titles \normalfont\textit{(Alt Bölüm Başlıkları)}}

Subtitle headings should be typed in Times New Roman 11\,pt, italic,
only first letter big, left justified, single line spacing, 12\,pt
before and 6\,pt after.

% ----
\subsection{Tables \normalfont\textit{(Tablolar)}}

Tables should be placed within the specified page limits and numbered
appropriately. The table heading should be placed above the table
(9\,pt, label bold, 12\,pt before / 3\,pt after).

\begin{table}[htbp]
  \caption{Table title \textit{(English table title)}}
  \centering
  \fontsize{9}{11}\selectfont
  \begin{tabular}{ll}
    \toprule
    Column Header 1 & Column Header 2 \\
    \midrule
    Sample Text 1   & Sample Text 2   \\
    Sample Text 3   & Sample Text 4   \\
    \bottomrule
  \end{tabular}
\end{table}

% ----
\subsection{Figures \normalfont\textit{(Şekiller)}}

Figures should be prepared electronically and placed within specified
page boundaries. The figure caption appears below the figure (9\,pt,
label bold, 6\,pt above / 12\,pt below).

\begin{figure}[htbp]
  \centering
  % \includegraphics[width=\linewidth]{your_figure}
  \fbox{\rule{0pt}{2cm}\rule{0.8\linewidth}{0pt}} % placeholder
  \caption{The caption of the figure \textit{(English caption)}}
\end{figure}

% ----
\subsection{Equations \normalfont\textit{(Denklemler)}}

Equations should be placed on a separate line and numbered
consecutively in parentheses, right-aligned.

\begin{equation}
  a = b + c
\end{equation}

% ============================================================
\section{Reference Text and Citations}

References and citations should be prepared in accordance with the
Harvard (Author, date) system. As an example:
\parencite{Surname1Surname22013}. All sources must be cited in the
text \parencite{Surname1etal2010}.

Sources must be typed in 9\,pt, half-width, single-line spacing,
0.5\,cm hanging indentation \parencite{Yilmaz2012}.
